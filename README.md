# 卓越中古車 - 中古車買賣預約平台 (Car Shop Management System)

這是一個基於 **Spring Boot** 開發的中古車平台專題，旨在提供透明、便捷的車輛買賣與預約流程。

## 🚀 技術
- **後端**: Java 1.8, Spring Boot, Spring Data JPA
- **前端**: Thymeleaf 模板引擎, HTML/CSS (Bootstrap)
- **資料庫**: H2 Database / MySQL
- **工具**: Eclipse IDE, Maven, Git/GitHub

## 🌟 主要功能 
- **車輛展示**: 提供精選車款列表，包含價格、年份與詳細規格。
- **買賣服務**: 使用者可線上提交賣車需求。
- **預約賞車**: 線上預約看車時間與點位。
- **會員系統**: 支援使用者註冊、登入與個人預約紀錄管理。

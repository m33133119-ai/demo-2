卓越中古車 - Car Shop Management System 

這是一個基於 Spring Boot 開發的中古車管理與預約平台專題，旨在提供透明、便捷的車輛買賣與預約流程。 

🚀 技術 
本專案採用當前主流的後端開發技術： 
後端框架: Spring Boot (簡化配置、快速開發) 
資料存取: Spring Data JPA (自動化資料庫 CRUD 操作) 
樣板引擎: Thymeleaf (動態網頁渲染) 
資料庫: H2 Database (開發環境)  
依賴管理: Maven (自動化 JAR 包管理) 
前端設計: HTML5, CSS3 (Bootstrap 框架) 

🌟 主要功能  
車輛展示系統: 串接資料庫顯示最新車況、價格與規格。 
預約賞車服務: 使用者可選定時間並提交預約單。 
買賣估價申請: 提供車主線上提交賣車需求的功能。 
會員管理: 支援使用者登入、註冊及個人預約紀錄查詢。 

📂 專案結構說明 
src/main/java: 存放後端 Java 邏輯 (Controller, Service, Repository, Entity)。 
src/main/resources/templates: 存放 Thymeleaf HTML 網頁模板。 
src/main/resources/static: 存放 CSS、圖片及 JavaScript 檔案。 
pom.xml: 定義專案所需的 Maven 依賴。
